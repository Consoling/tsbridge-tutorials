HTML Cheatsheet for Beginners by AZSH

Tip: Use <!-- comment --> to add comments in your HTML code!
Tag Description Example
<!DOCTYPE html> Defines the document type (HTML5) <!DOCTYPE html>
<html> Root element of an HTML page <html lang="en">...</html>
<head> Contains meta information <head>...</head>
<title> Title of the document (shown in browser tab) <title>My Page</title>
<meta> Metadata (charset, viewport, etc.) <meta charset="UTF-8">
<body> Main content of the HTML document <body>...</body>
<h1> to <h6> Headings (h1 is largest) <h1>Heading</h1>
<p> Paragraph <p>Text here</p>
<a> Link to another page <a href="https://example.com">Visit</a>
<img> Image <img src="img.jpg" alt="desc">
<ul>, <ol>, <li> Lists: unordered, ordered, list item <ul>
  <li>Item</li>
</ul>
<div> Generic container <div>Content</div>
<span> Inline container <span>Text</span>
<input> Input field <input type="text" placeholder="Name">
<button> Clickable button <button>Click Me</button>
<form> Form for user input <form>...</form>
<table> Table structure <table>...</table>
<strong>, <em> Bold and italic text <strong>Bold</strong>, <em>Italic</em>
<br> Line break <br>
<hr> Horizontal rule (line) <hr>
<link> Link to external resource (CSS) <link rel="stylesheet" href="style.css">
<script> Embed JavaScript <script src="app.js"></script>
<nav> Navigation links <nav>...</nav>
<header>, <footer> Page header and footer <header>...</header>
<section>, <article> Sections and articles <section>...</section>
<aside> Sidebar content <aside>...</aside>
<audio>, <video> Embed audio/video <audio src="song.mp3" controls></audio>
Attributes to remember:
id, class, style, src, href, alt, type, placeholder, value, name, target="_blank"
